Psalm Ansible role
=========

If you are a PHP developer and you use Psalm.

Requirements
------------
  - `php > 7.0` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.psalm

Check if it's installed:
------
    psalm --version
