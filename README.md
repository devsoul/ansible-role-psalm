Psalm Ansible role
=========
[![Build Status](https://travis-ci.com/devsoul/ansible-role-psalm.svg?branch=master)](https://travis-ci.com/devsoul/ansible-role-psalm)

If you are a PHP developer and you use Psalm.

Requirements
------------
  - `php > 7.0` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.psalm

Check if it's installed:
------
    psalm --version
